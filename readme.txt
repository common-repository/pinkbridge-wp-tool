== Description ==

Pinkbridge DAM acts as the Digital Asset Management System, serving as the cornerstone for our comprehensive marketing software "PinkBridge", enabling you to conveniently centralize all your digital files in one location.

The Formular Integration plugin streamlines the process of incorporating dynamic and interactive forms, created within the Pink Bridge system, into WordPress websites. This plugin enables users to effortlessly add pre-designed forms to WordPress pages using simple short codes. Seamlessly integrate forms created in Pink Bridge into your WordPress environment and enhance the functionality of your website with ease.

Key Features:
 
Effortless Form Integration: Easily add forms created in Pink Bridge to WordPress pages using intuitive shortcodes.
 
Dynamic Form Creation: Design custom forms within Pink Bridge's user-friendly interface using drag-and-drop components such as email fields, phone number fields, text inputs, Birth date and more.
 
Seamless Integration: Enjoy seamless integration between Pink Bridge and WordPress, ensuring a smooth user experience and consistent design across your website.
 
Enhanced Website Functionality: Elevate the functionality of your WordPress site by incorporating dynamic forms tailored to your specific needs, whether it's capturing user information, conducting surveys, or facilitating customer feedback.
 
Responsive Design: Ensure that forms created with Pink Bridge are fully responsive, adapting seamlessly to various screen sizes and devices for optimal user experience.

 = How It Works =

1. Create Forms in Pink Bridge: Design custom forms effortlessly within the Pink Bridge system using its intuitive drag-and-drop interface and powerful form-building capabilities.

2. Generate Shortcodes: Once your forms are created, generate shortcodes within Pink Bridge for each form you wish to integrate into your WordPress website.

3. Add Shortcodes to WordPress: Paste the generated shortcodes directly into WordPress pages or posts where you want the forms to appear, effortlessly embedding them into your website's content.

4. Manage Forms: Manage and track form submissions conveniently through Pink Bridge, gaining insights into user interactions and improving your website's performance.

== Installation ==

= Installation from within WordPress =

1. Navigate to **Plugins > Add New**.
2. Search for **Pinkbridge WP Tool**.
3. Install and activate the Pinkbridge WP Tool plugin.

= Manual installation =

1. Upload the entire `pinkbridge-dam` folder to the `/wp-content/plugins/` directory.
2. Visit the **Plugins** section.
3. Activate the Pinkbridge WP Tool plugin.

== Frequently Asked Questions ==
 
= How do I use this plugin? =

* Integrate API Endpoint provided by your administrator.
* Upon successful entry of the API Endpoint, an authentication screen will appear. Input the provided Email address and Password. Upon successful authentication, a list of forms will be displayed.
* Access the plugin via WP Admin by navigating to the PinkBridge > Formular menu. Please ensure you have the necessary permissions to manage plugins on your site to access this screen.
* The plugin facilitates copying the form shortcode and pasting it into any page builder.
* Easily select any file from the DAM option within the Media Library window.

= How to uninstall the plugin? =
 
Simply deactivate and delete the plugin.
